# как отцентровать элемент

из родителя:
display: flex;
justify-content: center;

из самого элемента:
margin-left: auto;
margin-right: auto;

# параметры элемента

1. контент
в зависимости от box-sizing;
2. высота, ширина 
3. паддинги (если border-box, то паддинги включены в высоту)
4. бордер
5. марджин
